Gerakan Sholat
dalam Genggamanmu

Arahkan kamera ke marker untuk melihat visualisasi 3D rangkaian gerakan sholat, dari takbiratul ihram hingga salam.

Panduan Penggunaan

  1. Siapkan marker. Cetak gambar marker pada kertas dengan ukuran minimal A5 dan permukaan rata (tidak terlipat).
  2. Pastikan pencahayaan cukup. Gunakan ruangan dengan cahaya merata, hindari pantulan langsung dan bayangan tajam pada marker.
  3. Tekan tombol “Mulai AR” dan izinkan akses kamera saat diminta oleh browser.
  4. Arahkan kamera ke marker dengan jarak sekitar 20–60 cm dan sudut kemiringan tidak lebih dari 45° untuk hasil deteksi terbaik.
  5. Gunakan tombol navigasi (Sebelumnya / Selanjutnya) di bagian bawah layar untuk berpindah antar-gerakan sholat.
  6. Ketuk “Dengarkan Narasi” jika tersedia, untuk mendengarkan penjelasan audio dari setiap gerakan.

Tentang Aplikasi

Aplikasi ini merupakan media edukasi berbasis Augmented Reality untuk memvisualisasikan rangkaian gerakan sholat secara interaktif. Dibangun menggunakan MindAR sebagai mesin pelacakan marker dan Three.js sebagai mesin rendering 3D berbasis WebGL, aplikasi berjalan langsung pada browser tanpa memerlukan instalasi maupun lisensi komersial.

Dikembangkan melalui pendekatan Multimedia Development Life Cycle (MDLC) model Luther-Sutopo, sebagai kontribusi pada pengembangan media pembelajaran berbasis grafika komputer yang lebih konkret dan adaptif dibanding media konvensional.
